Search beans by name or type, then expand nodes to inspect the property tree. Use Copy Path beside a node to get a bean.property expression for the executor below.

Dynamic Java Executor

Use chained access like bean.property; for multi-line input, join statements with semicolons, and the last expression becomes the result (lines with only // are ignored). After typing ., completion is sourced from the same tree on the left through /inspect child nodes, with reflective methods like List.add appended. Assignment examples: bean.prop = value or bean.list[0] = value. ctx provides reflection helpers.
